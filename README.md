# YOGAlater JavaScript Version

This is a browser-based conversion of the original Tkinter application.

## Files

- `index.html` — page structure
- `styles.css` — layout, background, modal, and responsive styling
- `app.js` — assessment logic, validation, score, and results
- `background.png` — optional background image you provide

## Run locally

Open `index.html` in a browser.

For best results, use a local server:

```bash
python -m http.server 8000
```

Then open:

```text
http://localhost:8000
```

## GitHub Pages

Upload these files to a GitHub repository, enable GitHub Pages, and use the repository root as the publishing folder.

The browser cannot run Tkinter. This version recreates the interface with HTML, CSS, and JavaScript.
